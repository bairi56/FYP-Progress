
AI-Powered Interviewing System FYP Progress:

Scope of the project 
In scope 

A user-friendly interface will be developed for both recruiters and candidates. Recruiters will be able to post job descriptions and manage interviews, while candidates can register, submit their skills and experiences, and attend interviews. 

  Interviews will be conducted in the computer science academic domain. 
  
  The system will evaluate candidates’ answers to assess their relevance and quality. 
  
  Dynamic dialogue management will be ensured. 
  
  Detect facial expressions to understand emotions and confidence level. 
  
  The system will handle the scoring mechanism and rank the candidates, that will help the recruiters with shortlisting. 


Out of scope 

  This system will cover only the computer science domain in this project.  
  
  The system will handle only academic theoretical parts. It may not be able to evaluate complex coding solutions. 
  
  The implementation of this project will focus on specific markets and will not be able to handle multilingual support.


Completed Tasks:  
1. Natural Language Understanding (NLU):
   - Intent classification using DistilBERT.  
   - Entity recognition using RoBERTa.  

2. Question Generation:
   - Automated question generation leveraging LLMs (1.7B parameters).  

3. Authentication System:
   - Developed login and signup functionality for the web application.
