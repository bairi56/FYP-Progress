
AI-Powered Interviewing System FYP Progress:

# Project Scope

## In Scope

1. **User-Friendly Interface**  
   - A user-friendly interface will be developed for both recruiters and candidates.  
   - Recruiters can post job descriptions and manage interviews.  
   - Candidates can register, submit their skills and experiences, and attend interviews.  

2. **Domain-Specific Interviews**  
   - The interviews will focus exclusively on the computer science academic domain.  

3. **Candidate Evaluation**  
   - The system will evaluate candidates’ answers to assess their relevance and quality.  

4. **Dynamic Dialogue Management**  
   - The system will ensure dynamic dialogue management for a seamless interview experience.  

5. **Facial Expression Detection**  
   - Facial expressions will be analyzed to understand emotions and confidence levels.  

6. **Scoring and Ranking**  
   - The system will implement a scoring mechanism to rank candidates and assist recruiters with shortlisting.  

## Out of Scope

1. **Domain Restriction**  
   - This system will cover only the computer science domain.  

2. **Evaluation Limitations**  
   - The system will handle only academic theoretical parts and may not evaluate complex coding solutions.  

3. **Market and Language Constraints**  
   - The implementation will focus on specific markets and will not include multilingual support.  


# Project Progress

2. **Question Generation**

3. **Authentication System**
